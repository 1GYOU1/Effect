bxSlider - 기본 슬라이드 ①

class="bxslider-default" 앨리먼트 내에 data 추가하여 사용
  • data-mode : horizontal or vertical or fade
  • data-maxSlides : 최대 보여지는 개수
  • data-slideMargin : 슬라이드 사이 margin
  • data-delay : 슬라이드 딜레이
  • data-speed : 슬라이드 속도
  • data-auto : 슬라이드 자동슬라이드
  • data-pager : 슬라이드 하단 동그라미 버튼
  • data-controls : 슬라이드 컨트롤러
  • data-moves : 슬라이드 넘어가는 개수

bxSlider - 슬라이드 선언

js에서 선언해서 사용
$(function() {} 등 jquery load 후 사용해야 안전
형제 클래스 삭제 시, 전체로 잡으면 모든 슬라이드에서 삭제될 수 있어 특정 부모 잡아놓고 삭제해야 함
참고 bxslider githubhttps://github.com/stevenwanderski/bxslider-4

html

    <div class="slide-obj">
    <ul>
        <li><img src="//gscdn.hackers.co.kr/champ/img/china/event/2022/05/0401/pc/type1/v1/cont08_01_01.png" alt=""/></li>
        <li class="on"><img src="//gscdn.hackers.co.kr/champ/img/china/event/2022/05/0401/pc/type1/v1/cont08_01_02.png" alt=""/></li>
        <li><img src="//gscdn.hackers.co.kr/champ/img/china/event/2022/05/0401/pc/type1/v1/cont08_01_03.png" alt=""/></li>
    </ul>
    </div>

js

    var slider = $('.slide-obj > ul').bxSlider({
    mode: 'horizontal',
    maxSlides: 3,
    moveSlides: 1,
    controls: true,
    pager: false,
    auto: false,
    onSlideNext: function(el) {
        el.closest('.slide-obj').find('li').removeClass('on');
        el.next('li').addClass('on');
    },
    onSlidePrev: function(el) {
        el.closest('.slide-obj').find('li').removeClass('on');
        el.next('li').addClass('on');
    }
    });
